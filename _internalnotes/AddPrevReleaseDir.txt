When creating a new "previous release" directory on the server:

1. Create new directory, e.g.
    mkdir new.x/

2. Copy files to new directory:
    cp *.txt new.x/
    cp *.sh new.x/
    cp favicon.* new.x/
    cp .ftpquota new.x/

3. cd new.x/

4. mkdir Backups

5. mkdir Saved

5. Edit UpdateVersion.sh
     remove copying of prev releases (2.6 etc) from the script
     remove copying of .htaccess, google.* from the script
