Build a new version:
    1) Run servesplice
    2) Upload to server as usual; all should work

When adding a new release version (e.g. 2.8):

    1) Update servesplice with new version #

    2) Create a new sitemapgen.<vers>.xml that appends the appropriate vers to each url (e.g. 2.7)

    3) Uupdate serveprev with new previous version #

    4) Add a new sitemap file name to each of these sitemapindex files:
       - sitemapindex.doc.xml
       - sitemap.docstest.xml
       - sitemap.docsdev.xml
