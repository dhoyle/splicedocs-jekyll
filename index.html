Splice Machine Quick Start

This page helps you to quickly get started with Splice Machine:

  • Follow the First Steps to get off to a good start.
  • Review the list of Splice Machine Tutorials to help you with specific tasks.
  • You'll find links to Additional Help resources here.

First Steps

  1. If you're using our database-as-a-service, follow the steps in our initial setup tutorial to create and provision your Splice Machine cluster.

    If you're using our on-premise database, follow the installation steps for your platform.

  2. Once your database is up and running, you can use our Import TPCH Data tutorial to quickly import the TPCH-1 data into your database and start running queries against it.
  3. Review our Built-in System Procedures section, which features database functions with which you might not be familiar.

  4. Check out the tutorials listed in the table below, which walk you through working with many of the features of Splice Machine. Many of these tutorials step you through specific, downloadable code examples and configuration options.

    • If you're new to Splice Machine, we strongly recommend that you quickly become familiar with our splice> command line interface by spending a few minutes with this tutorial.
    • Importing data into your Splice Machine database can prove a bit tricky, so we encourage you to review our importing data tutorial before diving into that process.
    • We also have short tutorials that step you through connecting different Business Intelligence tools to Splice Machine, and writing code in various languages to work with Splice Machine.
  5. Once you've become comfortable with working with Splice Machine, explore some of our more advanced tutorials for tasks such as streaming data, working with the Internet of Things, Machine Learning, and analytical workloads.

We also encourage you to spend a few minutes reviewing our documentation summary page to optimize your use of our documentation.
