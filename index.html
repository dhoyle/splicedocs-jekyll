Welcome to Splice Machine!

Welcome to Splice Machine, the database platform for adaptive applications that manage operational processes. This site contains documentation for our products and links to other useful resources.

Our new Managed Database Service in the Cloud is now available: Start Here!

Splice Machine is a hybrid relational database platform that allows you to run transactional (OLTP) and analytical (OLAP) SQL workloads, concurrently, against shared data. To ensure that one does not interfere with the other, Splice Machine uses resource isolation and a ground-breaking MVCC technology for concurrent, non-locking, distributed ACID transactions.

With Splice Machine, you can manage operational processes, large-scale IoT infrastructures, and use analytical processing and machine learning to improve over time.

Splice Machine delivers an open-source data platform that incorporates the proven scalability of HBase™ and the in-memory performance of Apache Spark™. The cost-based optimizer uses advanced statistics to choose the best compute engine, storage engine, index access, join order and join algorithm for each task. In this way, Splice Machine can concurrently process transactional and analytical workloads at scale.

Our Products

The Splice Machine platform offered in two product versions:

  • You can configure and deploy our Managed Database Service in the Cloud in a matter of minutes: simply register, adjust a configuration sliders for your anticipated data size, cluster power, and backup frequency, and you're up and running. A free trial version means that you can get going very quickly and see for yourself how well Splice Machine can work for your company's needs.
  • You can deploy our On-Premise Database on your own cluster that is managed by Cloudera, MapR, or Hortonworks. We also offer a standalone, trial version that allows you to experiment with Splice Machine on MacOS, Linux, and CentOS computers.
