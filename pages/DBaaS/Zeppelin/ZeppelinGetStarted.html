Getting Started with Zeppelin

 

When you click the Notebook button in your Cluster Management dashboard, you land on the Zeppelin welcome page. To start using Zeppelin with your database service, you need to log in to your database by clicking the Login button.

Once you're logged into Zeppelin for your database, you'll see the Zeppelin dashboard. From there, you can run already-created notebooks, modify existing notebooks, and create new notebooks.

Setting SpliceMachine as the Default Interpreter

The first time you run a Zeppelin notebook, you need to make sure that the Zeppelin Splice Machine interpreter is configured as your default interpreter. This is a simple matter of moving our interpreter to the top of the list that displays when you first open

To bind the splicemachine interpreter as the default interpreter for this Zeppelin notebook, drag it to the top of the list, then click the Save button.
