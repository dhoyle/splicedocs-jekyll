Using Zeppelin to Load and Run TPCH-1 Queries

This topic walks you through running the TPCH-1 notebook created by Splice Machine, which performs the following operations:

  • Creates a schema in your database for the TPCH benchmark data.
  • Creates and populates the TPCH-1 tables.
  • Imports the TPCH-1 data into the new schema/tables.
  • Allows you to run any or all of the TPCH-1 benchmark queries.
