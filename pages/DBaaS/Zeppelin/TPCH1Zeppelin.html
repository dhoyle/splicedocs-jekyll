Using Zeppelin to Load and Run TPCH-1 Queries

This topic walks you through running the TPCH-1 notebook created by Splice Machine.

Our Getting Started with Zeppelin page provides a very brief overview of using Zeppelin; If you're new to Zeppelin, we strongly encourage you to visit the Zeppelin documentation site to learn about creating, modifying, and running your own Zeppelin notebooks.

Running the TPCH-1 Notebook

You can access this Zeppelin notebook by clicking the Basics (Spark) link under Zeppelin Tutorials on the Zeppelin Dashboard page:

The TPCH-1 notebook:

  • Creates a schema and tables in your database for the TPCH benchmark data.
  • Imports the TPCH-1 data into the new schema/tables.
  • Creates indexes in the TPCH-1 schema.
  • Collects statistics for the schema.
  • Allows you to run any or all of the TPCH-1 benchmark queries.

You can run the entire notebook in one step, or you can sequentially run each paragraph in the notebook.

The remainder of this topic walks you through running the notebook tasks sequentially.
