Using Zeppelin Notebooks

The topics in this guide are only relevant to users of our database-as-a-service product.

This guide helps you to get started with using Zeppelin notebooks to interact with your Splice Machine database.

Topic Description
Getting Started with Zeppelin A quick and simple tutorial that show you how you can use Zeppelin notebook to load and query data, and apply different visualizations to the results.
Running TPCH-1 Queries A tutorial that walks through setting up and loading the TPCH-1 data, and then running queries against that data.
