Cloud Manager User Registration

This page describes the Splice Machine Cloud Manager registration page.

As you can see, the registration process is familiar; you can use your Google or Amazon account information to register, or you can manually register:

  1. Enter your first and last name.
  2. Enter and then confirm the email address you want to use for logging into the Splice Machine Cloud Manager.
  3. Enter and then confirm the password you want to use for logging into the Splice Machine Cloud Manager.
  4. Click the Register button.

Once you've successfully registered, you'll land on the Create New Cluster screen; we've created tutorials to quickly walk you through that process:

  • If you're registering to create a free trial cluster to experiment with Splice Machine, see our Initial Setup topic.
  • If you're registering to create a free trial cluster to experiment with Splice Machine, see our Free Trial Setup topic instead.
