Documentation Summary

This topic helps orient you to the Splice Machine documentation, in these sections:

  • Splice Machine Layout and Navigation explains the layout of our documentation web and shows you how to use the various navigation tools to land on the pages in which you're interested.
  • Summary of the Main Documentation Sections summarizes and links to all of the top-level sections in the Splice Machine documentation suite.

Splice Machine Layout and Navigation

Here's an image of the top portion of our documentation screen, which is called the topbar:

The topbar features these elements:

  • Click the Splice Machine Documentation title to return you to the home page of our documentation.
  • Click the Navigation Toggle to toggle the sidebar off (to expand the width of the main content) or off.
  • The top navigation menus includes our three main menus, which link to the main topic sections.
  • A sidebar for navigating to topics within the main sections.
  • A search bar that you can use to search for topic titles within the documentation.
  • The main content area, which contains the content of each topic.

The Top Navigation Menus

The Splice Machine documentation is divided into three major sections, each of which is a top-level menu:

Menu Description
The Splice Machine Menu links to the documentation categories that apply to all Splice Machine products.
The DB-Service Only Menu links to the documentation categories that apply only to our Cloud Managed, Database-as-Service product.
The On-Premise-DB Only Menu links to the documentation categories that apply only to our On-Premise-DB product.

When you click a section name link in any of the top-level menus, the sidebar changes to reflect the subsections and topics in that section.

The Navigation Toggle

The navigation toggle looks like this:

It defaults to being on, which means that the sidebar is displayed on the left side of the content area. You can click the switch to toggle the sidebar display off, which allows the content to fill the width of the window. Toggle the sidebar back on as necessary.

The Navigation Side Bar

The navigation sidebar is displayed on the left side of the window. Its content changes as you navigate through the sections of the documentation.

You can click the triangles in the sidebar to expand (or collapse) a topic area:

The content of the sidebar changes whenever you select a menu choice in the topbar.

The Search Bar

You can search for documentation topics with the searchbar:

Type topic titles or partial titles into the searchbar to display a list of links to topics in the documentation:

As you add characters, the list of topics narrows or expands. Click any of the topic links to land on that page.

The Main Content Area

The main content area displays each topic to which you navigate.

Summary of the Main Documentation Sections

The Splice Machine Documentation is divided into three main areas, each of which contains multiple sections, as shown in the following table:
