Documentation Summary

The Splice Machine Documentation is separated into these books:

Section Description
Installation Guide

This Guide provides step-by-step instructions for installing Splice Machine on compatible platforms, including:

  • The Splice Machine AWS Sandbox
  • Cloudera
  • Hortonworks
  • MapR
  • The Splice Machine Standalone version
Installation is only relevant for users of our on-premise database product; our database-as-a-service does not require installation, so only applies to our on-premise database product.
Command Line (splice>) Reference The reference manual for our command line interface.
SQL Reference The complete reference for Splice Machine SQL.
Developers

Contains a number of sections for database developers, including:

  • Developer Fundamentals contains topics describing the fundamentals of developing your database with Splice Machine.
  • Ingesting and Streaming Data walks you through importing data into your Splice Machine database and contains a number of short tutorials that help you to use various streaming tools with Splice Machine.
  • Connecting Your Database shows you how to connect to use our JDBC and ODBC drivers to connect Splice Machine with other data sources, and to connect specific Business Intelligence tools to Splice Machine.
  • Functions and Stored Procedures provides you with information about developing and using stored procedures and functions with Splice Machine.
  • Tuning and Debugging contains topics that help you to optimize and debug your queries.
Administrators

The Administrator's Guide contains a number of sections for database administrators, including:

  • Starting Your Database
  • Backing Up and Restoring
  • Cleaning Your Database
  • Shutting Down Your Database
The Splice Machine Management Console Describes and shows you how to use the Splice Machine Management Console, which is a browser-based tool for monitoring and reviewing the performance of your queries.
Notes

Release notes, glossary, license information, and other noteworthy information.

Typographic Conventions Used in Our Books

We use the following style conventions in the Splice Machine books:

Style Usage
Code font

Fixed width font used for all code and any literal information that you might need to enter. For example:

A table named Players with primary key player_id
Application command

Anything that represents a specific application or is displayed by a specific application. For example:

hbase -> Actions -> Stop
hdfs1 -> Actions -> Stop
zookeeper1 -> Actions -> Stop
Example

Code and command examples. For example:

splice> CREATE TABLE NewPlayers
   AS SELECT * 
         FROM Players WITH NO DATA;
0 rows inserted/updated/deleted
Highlighted text
(Highlighted code font)

Text that is a placeholder, showing you that you need to substitute your own value for the highlighted portion. For example:

Maximum Client Connections = 0
Maximum Session Timeout    = 120000
Shell command

Text displayed by an operating system. For example:

sudo -su hdfs hadoop fs -rm -r /hbase

Tips for Navigating the Documentation

The Splice Machine documentation uses a navigation model that allows you to see more content at one time while still providing a full menu of the topics in the entire documentation suite.

Tutorial topics are prefaced with a * in the table of contents.

Here's the top-level view of our database-as-a-service product documentation:

Searching

You can enter search terms into the search bar at the top-right of the Splice Machine documentation window to find matching topics. The search bar looks like this:

Enter your search term(s) and click the search icon. Our search algorithm finds any matches that are least four characters long, ranks them by relevance, and displays a search results page that includes a brief description of each page that contains matching text. You can click any found result to view that page in the documentation:

Contents / Index Navigation Pane:

The navigation pane provides two ways to select topics to view. When is displayed, you see the table of contents for our documentation. You can click a book node to expand or contract it, and you can click a topic node to display that topic. You can also click the small Expand/Collapse arrow to hide or display the navigation pane.

When Index is displayed, you see all of the index entries for our documentation. You can click a book node to expand or contract it, and you can click a topic node to display that topic. You can enter search terms to find index entries. You can also click the small Expand/Collapse arrow to hide or display the navigation pane.

Printing

You can click the printer icon while viewing any topic to print that topic.
