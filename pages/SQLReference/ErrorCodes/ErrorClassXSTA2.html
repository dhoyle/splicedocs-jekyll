Error Class XSTA2: XACT_TRANSACTION_ACTIVE

Error Class XSTA2: XACT_TRANSACTION_ACTIVE
SQLSTATE Message Text
XSTA2.S A transaction was already active, when attempt was made to make another transaction active.
