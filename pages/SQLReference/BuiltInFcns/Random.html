RANDOM

The RANDOM function returns a random number.

The RANDOM function returns a DOUBLE PRECISION number with positive sign, greater than or equal to zero (0), and less than one (1.0), given an INTEGER seed number.

Syntax

RANDOM()

Example

splice> VALUES RANDOM();
1
----------
0.2826393098638572
