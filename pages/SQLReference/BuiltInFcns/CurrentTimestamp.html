CURRENT_TIMESTAMP

CURRENT_TIMESTAMP returns the current timestamp.

This function returns the same value if it is executed more than once in a single statement, which means that the value is fixed, even if there is a long delay between fetching rows in a cursor.

Syntax

CURRENT_TIMESTAMP

or, alternately

CURRENT TIMESTAMP

Results

A timestamp value.

Examples

splice> VALUES CURRENT_TIMESTAMP;
1
-----------------------------
2015-11-19 11:03:44.095

1 row selected
