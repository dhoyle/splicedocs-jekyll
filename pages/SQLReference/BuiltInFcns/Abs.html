ABS or ABSVAL

ABS or ABSVAL returns the absolute value of a numeric expression.

Syntax

ABS(NumericExpression)

NumericExpression

A numeric expression; all built-in numeric types are supported: DECIMAL, DOUBLE PRECISION, FLOAT, INTEGER, BIGINT, NUMERIC, REAL, or SMALLINT

Results

The return type is the type of the input parameter.

Example

splice> VALUES ABS(-3);
1
----------
3
