Welcome to the Splice Machine On-Premise Database!

Welcome to Splice Machine, the database platform for adaptive applications that manage operational processes. This site contains documentation for our Managed Database Service in the Cloud.

Splice Machine is a hybrid relational database platform that allows you to run transactional (OLTP) and analytical (OLAP) SQL workloads, concurrently, against shared data. To ensure that one does not interfere with the other, Splice Machine uses resource isolation and a ground-breaking MVCC technology for concurrent, non-locking, distributed ACID transactions.

With Splice Machine, you can manage operational processes, large-scale IoT infrastructures, and use analytical processing and machine learning to improve over time.

Splice Machine delivers an open-source data platform that incorporates the proven scalability of HBase and the in-memory performance of Apache Spark. The cost-based optimizer uses advanced statistics to choose the best compute engine, storage engine, index access, join order and join algorithm for each task. In this way, Splice Machine can concurrently process transactional and analytical workloads at scale.

On-Premise Database Documentation

This sites includes documentation for our On-Premise Database, organized in the following books:

  • The Administration Guide describes the administrative tasks associated with configuring and maintaining your Splice Machine software, including topics such as starting and stopping your database, cleaning your database, and backing up.
  • Our Installation Guide walks you through the steps to install Splice Machine on your MacOS or Linux computer.
  • The SQL Reference Manual is a complete reference for our implementation of ANSI SQL.
  • Our splice> Command Line Reference Manual is a complete reference for our command line interpreter, which allows you to run SQL from the command line as well as taking advantage of a number of other commands for viewing and interacting with your database.

  • Our Developer's Guide contains numerous topics for developers working with our platform, in sections including Fundamentals, Connecting Programmatically, Connecting to BI Tools, Ingesting and Streaming Data, Analytics and Machine Learning, and Tuning Your Queries. 

  • Our Tutorials Collection contains a number of task-based tutorials to help you quickly take advantage of many Splice Machine features.
  • Our Release Notes section contains a release and update notes, requirements, and edition and license information.
