SYSCS_UTIL.SYSCS_KILL_TRANSACTION

The SYSCS_UTIL.SYSCS_KILL_TRANSACTION system procedure kills an active transaction. You can use this to kill stale transaction or to kill long-running queries.

Syntax

SYSCS_UTIL.SYSCS_KILL_TRANSACTION(id)

id

A long integer that specifies the transaction ID.

Results

This procedure does not return a result.

Example

splice> create index aidx on a(i);
ERROR SE001: Splice Engine exception: unexpected exception
ERROR XJ001: Java exceptions; 'There are active transactions [51, 52]:
   java.lang.RuntimeException'.

splice> call SYSCS_UTIL.SYSCS_KILL_TRANSACTION(51);
Statement executed.

splice> call SYSCS_UTIL_SYSCS_KILL_TRANSACTION(52);
Statement executed.
