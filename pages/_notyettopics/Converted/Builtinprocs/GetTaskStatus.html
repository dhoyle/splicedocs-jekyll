SYSCS_UTIL.SYSCS_GET_TASK_STATUS

The SYSCS_UTIL.SYSCS_GET_TASK_STATUS system procedure displays the status of running statements, including job information for each and the node on which each is running.

Syntax

SYSCS_UTIL.SYSCS_GET_TASK_STATUS()

Results

The displayed results of calling SYSCS_UTIL.SYSCS_GET_TASK_STATUS include these values:

Value Description
STATEMENT The SQL statement that started the job.
JOBID

The ID of the job(s) generated by the statement. Note that each job is a container of tasks.
JOBHOST The node on which the job is running.
TASKID The ID of the tasks contained in the job. Note that the tasks do the actual work required to complete the statement.
TASKHOST The node on which the task is running.
STATUS

The status of the task. This is one of the following values:

  • PENDING
  • RUNNING
  • FAILED
  • COMPLETED
