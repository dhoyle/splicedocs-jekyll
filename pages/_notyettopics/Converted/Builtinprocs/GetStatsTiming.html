SYSCS_GET_STATISTICS_TIMING

The SYSCS_UTIL.SYSCS_GET_STATISTICS_TIMING system procedure displays whether or not (1 or 0) timing statistics gathering is currently enabled.

Syntax

SYSCS_UTIL.SYSCS_GET_STATISTICS_TIMING()

Results

Displays a 1 if timing statistics gathering is currently enabled, and a 0 if not.

Example

CALL SYSCS_UTIL.SYSCS_GET_TIMING_STATISTICS();
