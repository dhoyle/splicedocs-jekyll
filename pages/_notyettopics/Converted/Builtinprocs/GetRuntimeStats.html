SYSCS_GET_RUNTIME_STATISTICS

The SYSCS_UTIL.SYSCS_GET_RUNTIME_STATISTICS system procedure displays whether or not (1 or 0) run-time statistics gathering is currently enabled.

Syntax

SYSCS_UTIL.SYSCS_GET_RUNTIME_STATISTICS()

Results

Displays a 1 if run-time statistics gathering is currently enabled, and a 0 if not.

Example

CALL SYSCS_UTIL.SYSCS_GET_RUNTIME_STATISTICS();
