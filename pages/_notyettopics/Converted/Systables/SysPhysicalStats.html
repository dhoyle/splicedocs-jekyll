SYSPHYSICALSTATS System Table

The SYSPHYSICALSTATS table describes the physical nodes in the cluster.

The following table shows the contents of the SYSPHYSICALSTATS system table.

SYSPHYSICALSTATS system table
Column Name Type Length Nullable Contents
HOSTNAME VARCHAR 32682 NO The hostname of the machine.
NUM_CPUS INTEGER 10 YES

The number of CPUs in the machine.
MAX_HEAP BIGINT 19 YES

The maximum heap size for the machine.
NETWORK_CONNS INTEGER 10 YES The number of network connections on the machine.
