SYSPRIMARYKEYS System Table

The SYSPRIMARYKEYS table describes the physical nodes in the cluster.

The following table shows the contents of the SYSPRIMARYKEYS system table.

SYSPRIMARYKEYS system table
Column Name Type Length Nullable Contents
CONSTRAINTID CHAR 36 NO The ID of the constraint.
