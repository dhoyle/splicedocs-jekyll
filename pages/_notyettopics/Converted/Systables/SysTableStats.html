SYSTABLESTATS System Table

The SYSTABLESTATS table contains data collected for a specific table in a specific partition on a specific conglomerate (ID used internally).

The following table shows the contents of the SYSTABLESTATS system table.

SYSTABLESTATS system table
Column Name Type Length Nullable Contents
CONGLOMERATEID BIGINT 19 NO The conglomerate ID.
PARTITION_ID VARCHAR 32672 NO

The partition ID.
LAST_UPDATED TIMESTAMP 29 NO

Timestamp of when this was last updated.
IS_STALE BOOLEAN 1 NO Whether this data is stale.
IN_PROGRESS BOOLEAN 1 NO Whether statistics are currently being gathered for this table.
ROWCOUNT BIGINT 19 YES The number of rows in the table.
PARTITION_SIZE BIGINT 19 YES The size of the partition.
MEANROWWIDTH INTEGER 10 YES The mean width of a row in the table.
