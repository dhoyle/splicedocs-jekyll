SYSBACKUPFILESET System Table

The SYSBACKUPFILESET table maintains information about the files involved in each backup job.

SYSBACKUPFILESET system table
Column Name Type Contents
BACKUP_ITEM VARCHAR The conglomerate number of the table.
REGION_NAME VARCHAR The name of a table region.
FILE_NAME VARCHAR The name of the file in which data is stored.
INCLUDE BOOLEAN Whether of not this file will be included in the next incremental backup.
