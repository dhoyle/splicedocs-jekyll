SYSCOLUMNSTATS System Table

The SYSCOLUMNSTATS table contains data collected for a specific column in a specific partition on a specific conglomerate (ID used internally).

The following table shows the contents of the SYSCOLUMNSTATS system table.

SYSCOLUMNSTATS system table
Column Name Type Length Nullable Contents
CONGLOM_ID BIGINT 19 NO The conglomerate ID.
PARTITION_ID VARCHAR 32672 NO

The partition ID.
COLUMN_ID INTEGER 10 NO

The column ID.
DATA CHAR -1 NO An object that contains the statistics objects.
