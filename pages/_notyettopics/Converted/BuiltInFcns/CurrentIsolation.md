# CURRENT\_ISOLATION

`CURRENT ISOLATION` returns the current isolation level as a char(2)
value of either \"\"(blank), \"UR\", \"CS\", \"RS\", or \"RR\".

## Syntax

<div class="fcnWrapperWide" markdown="1">
    CURRENT ISOLATION
{: .FcnSyntax}

</div>
## Example

<div class="preWrapper" markdown="1">
    VALUES CURRENT ISOLATION
{: .Example}
