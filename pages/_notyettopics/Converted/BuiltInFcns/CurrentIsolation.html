CURRENT_ISOLATION

CURRENT ISOLATION returns the current isolation level as a char(2) value of either ""(blank), "UR", "CS", "RS", or "RR".

Syntax

CURRENT ISOLATION

Example

VALUES CURRENT ISOLATION
