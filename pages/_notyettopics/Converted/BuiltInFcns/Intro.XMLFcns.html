XML Functions

This section contains the reference documentation for the XML functions built into Splice Machine SQL:

Function Name Description
XMLEXISTS An SQL/XML operator that you can use to query XML values in SQL.
XMLPARSE An SQL/XML operator that you can use to parse a character string expression into a Splice Machine XML value.
XMLQUERY An SQL/XML operator that you can use to query XML values in SQL.
XMLSERIALIZE An SQL/XML operator that you can use to convert an XML type to a character type.
