Bit Data Types

This section contains the reference documentation for the bit data types built into Splice Machine SQL:

Data Type Description
CHAR FOR BIT DATA Allows you to store byte strings of a specified length.
LONG VARCHAR FOR BIT DATA Allows storage of bit strings up to 32,670 bytes. It is identical to LONG VARCHAR FOR BIT DATA, except that you cannot specify a maximum length when creating columns of this type.
VARCHAR FOR BIT DATA Allows you to store binary strings less than or equal to a specified length. It is useful for unstructured data for which character strings are not appropriate, such as images.
