LONG VARCHAR FOR BIT DATA

The LONG VARCHAR FOR BIT DATA type allows storage of bit strings up to 32,670 bytes. It is identical to VARCHAR FOR BIT DATA, except that you cannot specify a maximum length when creating columns of this type.

Syntax

LONG VARCHAR FOR BIT DATA

JDBC Metadata Type (java.sql.Types)

LONGVARBINARY
