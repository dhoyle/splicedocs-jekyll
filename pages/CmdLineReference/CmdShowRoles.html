Show Roles

The show roles command a sorted list of all of the roles defined in the database.

Syntax

SHOW ROLES

Examples

splice> create role testRole;
0 rows inserted/updated/deleted
splice> show roles;

ROLEID
------------------------------
TESTROLE

1 row selected
