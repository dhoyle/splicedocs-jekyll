Show Schemas

The show schemas command displays all of the schemas in the current connection.

Syntax

SHOW SCHEMAS

Examples

splice> show schemas;
TABLE_SCHEM
------------------------------
NULLID
SPLICE
SQLJ
SYS
SYSCAT
SYSCS_DIAG
SYSCS_UTIL
SYSFUN
SYSIBM
SYSPROC
SYSSTAT
11 rows selected
