# Show Settable_Roles The show settable_roles command displays all of the roles that the current session can set, comprising all of the roles that have been granted to the current user or to PUBLIC. ### Syntax
SHOW SETTABLE_ROLES {: .FcnSyntax xml:space="preserve"}
### Examples
splice> SHOW SETTABLE_ROLES; ROLEID ----------------------------- ADMIN 1 row selected {: .AppCommand xml:space="preserve"}
