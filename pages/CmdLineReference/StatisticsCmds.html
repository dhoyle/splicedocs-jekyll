Query Optimization and Statistics Commands

This section describes the splice> commands you use to work with statistics and query optimization:

Command Description Usage
Analyze Collects statistics for a table or schema. splice> analyze table myTable;
splice> analyze schema myschema;
Explain Plan Displays the execution plan for an SQL statement. splice> explain plan select count(*) from si;
